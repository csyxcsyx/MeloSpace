# 步骤五前端主流程与播放器说明

> 状态：已完成。步骤五已完成前端主流程、全局播放器、沉浸式歌词页、LDDC 逐字歌词导入、管理员音乐与用户管理能力，并已通过前后端构建测试与远程推送。

## 1. 完成内容

- 新增 `music-web-frontend` 前端工程，采用 Vue 3、Vite、TypeScript、Pinia、Vue Router、Axios 和 lucide-vue-next。
- 项目正式命名为 `MeloSpace`，Web 标题、侧栏品牌、登录页和播放器默认文案已统一。
- 落地 Apple Music 风格启发的音乐应用界面：固定浅灰侧栏、玻璃拟态卡片、横向推荐卡、四列歌曲榜单和放大的底部播放器。
- 完成前端路由：`/discover`、`/search`、`/songs/:id`、`/playlists/:id`、`/me`、`/admin`、`/login`、`/player`。
- 完成 API 客户端封装，统一处理 `ApiResponse<T>`、分页结果、Token 注入、401 清理登录态和 403 权限提示。
- 完成登录/注册、发现页、搜索、歌曲详情、歌单详情、个人中心、后台管理、全局播放器和沉浸歌词页。
- 歌曲栏支持主流音乐软件交互：双击歌曲行播放并进入歌词页，单击封面播放/暂停，封面悬浮显示播放/暂停按钮，三点菜单支持收藏、添加到歌单和登录后下载。
- 全局播放器使用单个隐藏 `<audio>`，支持播放、暂停、上一首、下一首、进度跳转、音量调整、错误提示、本地状态保存和队列自动续播。
- 底部播放器封面悬浮时可进入全屏歌词页；歌曲自然播放结束后会继续播放当前队列下一首，队列末尾自动停止。
- 沉浸歌词页占满页面，包含返回/首页入口、唱片式封面、歌词滚动高亮、底部完整控制栏，并根据当前封面提取主题色。
- 最近播放接口返回歌曲摘要，个人中心和发现页可直接展示歌曲名称与歌手名。
- 开发期通过 Vite 代理 `/api` 和 `/media` 到 `http://localhost:8080`，无需改动后端 CORS。
- 管理员后台已支持 Web 端上传音频、专辑图和歌词文件，歌曲封面跟随专辑图；新增歌曲可绑定不同歌手归属的专辑，以支持合作曲。
- 管理员后台已支持编辑已上传歌曲和专辑信息，支持歌曲上下架、删除歌曲/专辑，以及查看用户列表并注销其他用户。
- 普通用户可在个人页注销自己的账号；后台用户管理仅展示账号名、昵称、角色、状态、注册时间和密码加密存储状态，不展示明文密码或密码哈希。
- 已接入本地 `chenmozhijin/LDDC` 源码辅助匹配逐字歌词，前端歌词面板按歌词文件时间滚动并使用渐变式逐字高亮。

## 2. 本地启动

后端：

```powershell
cd music-web-backend
.\mvnw.cmd spring-boot:run
```

前端：

```powershell
cd music-web-frontend
npm install
npm run dev
```

前端默认地址：`http://127.0.0.1:5173/`。

## 3. 当前功能门禁

- [x] 匿名用户可进入 `/discover` 浏览歌曲和公开歌单。
- [x] 双击歌曲行可播放并进入 `/player` 歌词页，音频加载失败时显示错误提示。
- [x] 单击歌曲封面可播放/暂停但不跳转，悬浮封面时显示播放/暂停按钮。
- [x] 歌曲行三点菜单可收藏、添加到我的歌单和登录后下载音乐。
- [x] 底部播放器封面悬浮时显示进入歌词页入口。
- [x] 当前队列中的歌曲自然播放结束后可自动进入下一首，队列末尾停止。
- [x] 用户可登录/注册，登录态写入 `localStorage`。
- [x] 登录用户可收藏歌曲/歌单、评论歌曲、创建歌单、向歌单添加歌曲并查看个人中心。
- [x] 个人中心和发现页最近播放显示歌曲名与歌手名。
- [x] 管理员可进入 `/admin` 查看统计、上传资源、创建歌手、创建歌曲和切换歌曲上下架状态。
- [x] 页面在宽屏、窄屏和移动端有响应式布局，桌面侧栏固定，放大后的播放器不会遮挡主体操作。
- [x] 主要卡片统一圆角与玻璃风格，播放器、歌词页和歌曲列表有清晰 hover/focus/当前播放态。
